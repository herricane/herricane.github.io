## 为什么选择 Astro + Fuwari

搭这个博客之前看了不少方案，最终选了 **Astro** + **Fuwari 主题**，理由很简单：

1. **Markdown 写文章** — 所有文章都是纯 Markdown，用 Git 管理版本
2. **Giscus 评论** — 基于 GitHub Discussions，不需要第三方评论服务
3. **极快的加载速度** — Astro 默认输出零 JS 的静态 HTML
4. **漂亮的中文排版** — Fuwari 对中文的支持很好
